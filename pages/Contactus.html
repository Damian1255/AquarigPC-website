Troubleshooting Tips

PC is not starting

- Check if power cable is connected and power outlet is turned on

- Check if power supply is switch is on

PC is on but no display output

- Check if DP/HDMI/VGA cable is connected to the Computer

- Check if monitor is turned on

- Change display cable
